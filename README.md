# 📊 Electronic Product Analysis Dashboard (Power BI + EDA)

An interactive **Power BI dashboard** built for analyzing electronic product data using **data cleaning, exploratory data analysis (EDA), and visual storytelling**.

---

## 🚀 Project Overview

This project focuses on:
- Data Cleaning & Preparation
- Exploratory Data Analysis (EDA)
- Interactive Dashboard Development in Power BI
- Business Insights from product data

---

## 📁 Dataset Used

- `electronic_cleaned.csv` → Main dataset for analysis  
- `product_cleaned.csv` → Product reference dataset  

---

## 📊 Dashboard Features

✔ Interactive Slicers (Category Filter)  
✔ KPI Cards (Total Products, Total Records, Sales)  
✔ Category-wise Analysis  
✔ Pie/Donut Chart Distribution  
✔ Bar Chart Insights  
✔ Data Table View
